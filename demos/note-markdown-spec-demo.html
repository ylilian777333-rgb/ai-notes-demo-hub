# 产品评审会纪要
2026-04-13 · 参会人：张三、李四、王五、赵六

## 会议共识

- 首页采用**A方案（卡片流）**，合集入口改为横滑卡片
- AI推荐tab点击率**12%**，目标30%，本周出方案
- 录音转写准确率**85%**，接入自定义词库

## 关键决策

| 议题 | 决策 | 负责人 |
|------|------|--------|
| 首页改版 | 采用A方案，优化合集入口 | 赵六 |
| AI推荐 | 出混合排序方案 | 李四 |
| 录音转写 | 接入自定义词库 | 王五 |

## 议题详情

### 1. 首页改版方案

设计提供了两版方案：

- **A方案**：保留卡片流，用户已习惯
- **B方案**：改为信息流，改动成本高

> 💡 最终决定采用A方案，合集入口改为横滑卡片。

### 2. AI推荐算法

当前点击率**12%**，目标**30%**。

> 方案：编辑时间+内容相似度混合排序

### 3. 录音转写准确率

准确率约**85%**，专有名词识别错误。

## 待办事项

- [ ] 李四：出AI推荐排序方案 📅 周五前
- [ ] 王五：完成自定义词库接入 📅 两周内
- [x] 赵六：完成合集入口设计稿 📅 下周一

## 日程

- 📅 下周二 线下评审会

# 首页可以加个AI推荐区
2026-04-12 12:30 · 语音速记 · 0:42

走路时想到的，首页除了时间线还可以有一个
AI推荐区，根据用户最近的笔记内容推荐相关
的历史笔记。

比如用户刚写了一篇会议纪要，推荐区可以展
示上次同主题的纪要。

## 关键想法

- 首页增加**AI推荐区**
- 基于**最近编辑内容**推荐历史笔记
- 推荐区展示**同主题/同项目**的关联笔记

## 待办

- [ ] 和设计讨论推荐区的位置和样式
