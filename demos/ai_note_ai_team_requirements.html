📋 一、笔记生成模板清单（共14套）
⚠️ 回应AI同学问题：「多少个模板的支持？」—— 共14种笔记类型，每种需要一套独立的 Prompt + 后处理逻辑。建议分两期落地。
# 笔记类型 触发场景 输入 → 输出 处理方式 优先级
1 📋 会议纪要 长录音（>5min）+ 日历有会议 会议录音 → 参会人 + 议题 + 结论 + 行动项 全自动 P0
2 💡 灵感速记 短语音（<1min）+ 无上下文 短语音 → 核心想法 + 关联 + 下一步 全自动 P0
3 ✅ 任务清单 包含多个事项/时间的语音 语音 → 优先级清单 + 分类 + 时间 全自动 P0
4 📞 通话摘要 通话录音（来自通讯助手） 通话录音 → 通话目的 + 结论 + 承诺事项 全自动 P0
5 📝 课堂笔记 长录音 + 无会议上下文 课堂录音 → 知识点结构 + 重点标注 + 举例 全自动 P0
6 📸 截图整理 截图/图片输入 截图 → 结构化信息 + 待办提取 + 原图关联 全自动 P0
7 📰 文章摘要/翻译 URL / 英文文章分享 网页链接 → 核心观点 + 关键数据 + 一句话总结 全自动 P1
8 🗂️ 文档摘要 PDF / Word / PPT 文件 文件 → 核心内容 + 结构层级 + 关键标注 全自动 P1
9 ✉️ 邮件/文案草稿 明确的创作指令 用户指令 → 称呼 + 正文 + 结尾 对话式 P1
10 📊 研究简报 用户指令 + 笔记库素材 指令 → 背景 → 分析 → 发现 → 结论 → 建议 对话式 P1
11 📹 视频提取 视频文件/链接 视频 → 核心要点 + 时间线 + 关键帧 全自动 P2
12 📅 周报/日报 周报指令 + 本周笔记数据 本周数据 → 按项目：进展 + 计划 + 风险 半自动 P2
13 🔖 收藏整理 跨应用收藏的文章/链接 收藏列表 → 排序清单 + 一句话摘要 + 分类标签 全自动 P2
14 🌐 每日简报 系统自动生成（日历+待办+资讯） 多源数据 → 日程 + 待办 + 资讯 + 统计 全自动 P2
✅ 建议分期：一期（Beta前）落地 P0 的 6 套模板，二期补齐 P1 的 4 套，P2 的 4 套后续迭代。
🧠 二、场景自动识别能力（AI团队负责）

场景判断逻辑

用户输入内容后，AI需要自动判断属于哪种场景，选择对应模板生成笔记。判断依据三个维度：

  • 输入形式：语音/图片/文件/链接/文字指令
  • 内容长度：短语音（<1min）vs 长录音（>5min）
  • 上下文线索：日历是否有会议、来源APP、用户历史行为
用户行为 输入特征 AI判断为 生成笔记类型
开完会，录音结束 长录音（>5min）+ 日历有会议 会议场景 会议纪要
随口说了一段话 短语音（<1min）+ 无上下文 灵感场景 灵感速记
截了一张图 截图/图片 截图场景 截图整理
分享了链接/文件 URL / PDF / 文档 文档场景 文章摘要 / 文档摘要
说"帮我写封邮件" 明确的创作指令 创作场景 邮件草稿
说"今天要做这些事" 包含多个事项/时间 任务场景 任务清单
接完一个电话 通话录音（来自通讯助手） 通话场景 通话摘要
上课/听讲座时录音 长录音 + 无会议上下文 学习场景 课堂笔记
⚠️ 判断优先级：用户明确说了要干嘛 > 有上下文线索（日历/来源APP） > 根据输入形式和长度猜测。判断不了时直接问用户。
🧩 三、记忆体系需求（越用越懂你）
⚠️ 回应AI同学问题：「是否要实现记忆？」—— 是的，记忆是核心差异化能力。建议分两阶段实现。

一期·基础记忆 4/20 Beta前必须

  • 写作风格偏好（正式/口语化）
  • 详细程度偏好（精简/详细）
  • 结构偏好（列表 vs 表格 vs 段落）
  • 常用术语和缩写记忆

实现方式：记录用户每次修改行为，提取偏好规则，下次生成时注入

二期·深度记忆 Beta后持续迭代

  • 人物关系图谱（张总=产品总监）
  • 项目上下文关联（跨笔记关联）
  • 历史笔记内容检索和引用
  • 重复内容检测和提示
  • 风格一致性（和历史笔记保持一致）

实现方式：构建用户知识图谱 + 向量检索 + 偏好模型

AI记住什么 怎么学到的 下次怎么用
写作风格 用户总是把正式表达改成口语化 下次生成时默认用口语化风格
详细程度 用户经常删掉细节，只保留结论 下次生成时更精简，结论先行
人物关系 多次出现"张总=产品总监" 下次提到张总时自动标注职位
常用术语 用户多次使用"大招"指代某功能 下次遇到"大招"自动理解含义
结构偏好 用户总是把列表改成表格 下次优先用表格呈现
🎯 四、AI输出质量标准（7条通用标准）

优先级排序：信息保真 > 结构匹配 > 信息密度 > 可读性 > 可执行性 > 克制 > 上下文感知

前四条是基础，任何笔记都必须满足；后三条是进阶，随记忆体系成熟逐步提升。

序号 标准 核心要求 严重等级
1 信息保真 关键事实不丢失 · 不产生幻觉 · 不篡改立场 · 来源可追溯 🔴 致命
2 结构匹配 场景识别正确 · 模板选择正确 · 必要元素齐全 · 层级清晰 🔴 致命
3 信息密度 冗余过滤 · 提炼而非罗列 · 长度匹配输入 · 零废话 🟡 重要
4 可读性 标题准确 · 语言通顺 · 逻辑连贯 · 术语保留 🟡 重要
5 可执行性 待办三要素（做什么+谁做+什么时候）· 时间可解析 · 格式可操作 🟡 重要
6 克制 不过度发散 · 不擅自决策 · 建议与事实分开 · 语气匹配 🟡 重要
7 上下文感知 关联已有内容 · 风格一致 · 术语对齐 · 重复检测 🟢 优化
